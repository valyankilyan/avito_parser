# avito_parser

### Что делает:
- Собирает цены товаров со страничек, лежащих в pages

### Требования:
- Версия Python - от 3.6.x

### Перед запуском:

#### Установить все библиотеки
`pip3 install -r requirenments.txt`

#### Сохранить странички в папочку pages

#### Запустить скрипт
`python3 avito.py`
